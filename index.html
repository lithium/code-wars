LD

        
Load data into memory

        

          ld address, value
        

        

          
The memory location referenced by address will be changed to value.

        

        
Examples

        

          
 ld (-1), 0 

          
load previous memory location with 0

        

        

          
 ld @label, (-5) 

          
load the memory location at label with the data at memory location -5

        

        

          
 ld 5, 0x42 

          
You cannot reference absolute memory locations.
