MOV

Change memory by MOVing data into it.

address = value
RedCode Equivalent
MOV value, address

The memory location referenced by address will be changed to value.

Examples
mov 0, (-1)
change previous memory location to 0
mov (-5), @label
change the memory location at label to be the data stored at memory location -5
ld 0x42, 5
You cannot reference absolute memory locations.

.DATA

Data constant

.DAT value
.DATA value

The 16-bit value will be included at the current location in the compiled program.

Examples
.DAT 0
.DATA 0xfe000f2a

if ==

Compare and If Equal run next instruction

if a == b:
  <next instruction>
RedCode Equivalent
SNE value, value
CMP value, value

Compare the 12-bit value a and the 12-bit value b. If they are equal, run <next instruction>. Otherwise, skip it.

Examples
loop: counter -= 1
if counter == 0:
  goto loop
if *foo == 42:

if !=

Compare and Not If Equal run next instruction

if a != b:
  <next instruction>
RedCode Equivalent
SEQ value, value

Compare the 12-bit value a and the 12-bit value b. If they are not equal, run <next instruction>. Otherwise, skip it.

Examples
loop:
if *label != 42:
  label += 1
goto loop

if >

If Greater Than, run next instruction

if a > b:
  <next instruction>
if b <= a:
  <next instruction>

If the 12-bit value a is greater than the 12-bit value b, run <next instruction>. Otherwise, skip it.

Examples
if counter > 100:
  counter = 100

if <

If Greater than or equal to, run next instruction

if a < b:
  <next instruction>
if b >= a:
  <next instruction>

If the 12-bit value a is less than the 12-bit value b, run <next instruction>. Otherwise, skip it.

Examples
loop:
if *label != 42:
  label += 1
goto loop

ADD

Change a memory location by adding a value.

dest += value
RedCode Equivalent
ADD value, dest
Description

The 15-bit value will be added to the value at location dest

Examples
(5) += 23
add 23 to the memory location at +5
total += subtotal
add the value in memory at the label subtotal to the memory location at referenced by the label total
